✅ Prérequis

Vérifications avant installation

  • ✅ Windows 10 ou supérieur
  • ✅ Droits administrateur sur la machine
  • ✅ Minimum 2 GB d'espace disque disponible
  • ✅ Ports 80 (Apache) et 3306 (MySQL) disponibles
  • ✅ Pare-feu Windows configuré (autorisations nécessaires)

Désactiver les services conflictuels

⚠️ Important : Si vous avez déjà un serveur web ou MySQL installé (IIS, Apache, MySQL), vous devez les arrêter ou les désinstaller pour éviter les conflits de ports.
1 Désactiver IIS (si installé)
  1. Ouvrir le Panneau de configuration
  2. Aller dans Programmes et fonctionnalités
  3. Cliquer sur Activer ou désactiver des fonctionnalités Windows
  4. Décocher Internet Information Services
  5. Redémarrer l'ordinateur
2 Vérifier les ports disponibles

Ouvrir PowerShell ou CMD en tant qu'administrateur :

# Vérifier si le port 80 est utilisé netstat -ano | findstr :80 # Vérifier si le port 3306 est utilisé netstat -ano | findstr :3306
ℹ️ Si ces commandes retournent des résultats, un service utilise déjà ces ports. Vous devrez l'arrêter.

🔧 Installation de XAMPP sur Windows

1 Téléchargement de XAMPP
  1. Aller sur le site officiel : https://www.apachefriends.org/
  2. Cliquer sur XAMPP for Windows
  3. Télécharger la version 8.2.12 (ou la plus récente)
  4. Sauvegarder le fichier xampp-windows-x64-8.2.12-0-VS16-installer.exe
2 Lancement de l'installation
  1. Double-cliquer sur le fichier téléchargé
  2. Si Windows affiche un avertissement de sécurité, cliquer sur Oui
  3. Si l'UAC (User Account Control) demande confirmation, cliquer sur Oui
⚠️ Attention : Si vous voyez un message concernant l'antivirus, il est recommandé d'ajouter XAMPP aux exceptions de votre antivirus pour éviter les problèmes de performance.
3 Configuration de l'installation
  1. Écran de bienvenue : cliquer sur Next
  2. Sélectionner les composants à installer :
    • ☑ Apache
    • ☑ MySQL
    • ☑ PHP
    • ☑ phpMyAdmin
    • ☐ Perl (optionnel)
    • ☐ Tomcat (optionnel)
    • ☐ Webalizer (optionnel)
  3. Cliquer sur Next
  4. Choisir le dossier d'installation (par défaut : C:\xampp)
  5. Cliquer sur Next
  6. Décocher Learn more about Bitnami (optionnel)
  7. Cliquer sur Next
  8. Cliquer sur Next pour commencer l'installation
ℹ️ Recommandation : Utilisez le dossier d'installation par défaut C:\xampp pour simplifier la configuration.
4 Finalisation de l'installation
  1. Attendre la fin de l'installation (2-3 minutes)
  2. Cocher Do you want to start the Control Panel now?
  3. Cliquer sur Finish
✅ XAMPP est maintenant installé !

🔥 Configuration du Pare-feu Windows

⚠️ Au premier lancement d'Apache et MySQL, Windows Firewall vous demandera d'autoriser l'accès réseau. Acceptez ces autorisations.

Si le pare-feu bloque Apache ou MySQL

1 Ouvrir les paramètres du pare-feu
  1. Ouvrir le Panneau de configuration
  2. Aller dans Système et sécurité → Pare-feu Windows Defender
  3. Cliquer sur Paramètres avancés (à gauche)
2 Créer des règles entrantes
  1. Cliquer sur Règles de trafic entrant
  2. Cliquer sur Nouvelle règle... (à droite)
  3. Sélectionner Programme → Next
  4. Choisir Ce chemin d'accès du programme : C:\xampp\apache\bin\httpd.exe
  5. Sélectionner Autoriser la connexion → Next
  6. Cocher Domaine, Privé, Public → Next
  7. Nom : Apache XAMPP → Terminer

Répéter pour MySQL avec le chemin : C:\xampp\mysql\bin\mysqld.exe

🎮 Utilisation du XAMPP Control Panel

1 Démarrer le Control Panel
  1. Double-cliquer sur l'icône XAMPP Control Panel sur le bureau
  2. Ou lancer depuis : C:\xampp\xampp-control.exe
  3. Clic droit → Exécuter en tant qu'administrateur (recommandé)
2 Démarrer les services
  1. Cliquer sur Start à côté d'Apache
  2. Cliquer sur Start à côté de MySQL
✅ Les modules doivent s'afficher en vert avec la mention "Running"

Boutons utiles du Control Panel

Bouton Description
Start/Stop Démarrer ou arrêter un service
Admin Ouvrir l'interface web (Apache) ou phpMyAdmin (MySQL)
Config Modifier les fichiers de configuration
Logs Consulter les logs d'erreur
Shell Ouvrir un terminal avec les outils XAMPP
Netstat Voir les ports utilisés

Vérifier l'installation

  1. Ouvrir un navigateur web
  2. Aller sur : http://localhost
  3. Vous devriez voir la page d'accueil XAMPP
  4. Tester phpMyAdmin : http://localhost/phpmyadmin

🗄️ Configuration de la Base de Données

1 Accès à phpMyAdmin
  1. S'assurer que MySQL est démarré dans XAMPP Control Panel
  2. Ouvrir un navigateur
  3. Aller sur : http://localhost/phpmyadmin
  4. Se connecter (par défaut : utilisateur root, mot de passe vide)
2 Création de la base de données
  1. Dans phpMyAdmin, cliquer sur l'onglet Bases de données
  2. Dans "Créer une base de données" :
    • Nom : gestioncharge_mosca
    • Interclassement : utf8mb4_general_ci
  3. Cliquer sur Créer
3 Création des tables via SQL
  1. Sélectionner la base gestioncharge_mosca dans la liste de gauche
  2. Cliquer sur l'onglet SQL
  3. Copier-coller les requêtes SQL suivantes

Table Utilisateurs

CREATE TABLE Utilisateurs ( id INT AUTO_INCREMENT PRIMARY KEY, nom VARCHAR(100) NOT NULL, prenom VARCHAR(100) NOT NULL, email VARCHAR(150) NOT NULL UNIQUE, mot_de_passe VARCHAR(255) NOT NULL, role ENUM('admin', 'technicien', 'utilisateur') DEFAULT 'utilisateur', date_creation TIMESTAMP DEFAULT CURRENT_TIMESTAMP, derniere_connexion TIMESTAMP NULL, actif BOOLEAN DEFAULT TRUE, INDEX idx_email (email), INDEX idx_role (role) ) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4 COLLATE=utf8mb4_general_ci;

Table Donnees

CREATE TABLE `Donnees` ( `Processus` varchar(40) NOT NULL, `Tache` varchar(200) NOT NULL, `Charge` decimal(15,2) DEFAULT NULL, `Date` date DEFAULT NULL ) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4 COLLATE=utf8mb4_general_ci;
